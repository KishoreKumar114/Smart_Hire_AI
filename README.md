# 🚀 SmartHire AI - Intelligent Recruitment Platform

> **AI-powered recruitment platform** that automates hiring workflows with resume screening, interview preparation, candidate management, and data-driven analytics.

---

## ✨ Features

### 🤖 **AI-Powered Recruitment**
| Feature | Description |
|---------|-------------|
| 📄 **Resume Screening** | Extract information from PDF/DOCX resumes with AI parsing |
| 🎯 **AI Interview Prep** | Generate domain-specific questions & answers |
| 📝 **JD Generator** | Create professional job descriptions instantly |
| 👥 **Candidate Matching** | AI-based match scoring (75-95% accuracy) |

### 📊 **Recruitment Management**
| Feature | Description |
|---------|-------------|
| 👤 **Candidate Database** | Track applicants with detailed profiles |
| 📅 **Interview Scheduler** | Send professional email invitations |
| 📈 **Analytics Dashboard** | Hiring trends, success rates, predictions |
| 📧 **Email Automation** | Send interview invites & follow-ups |

### 🎨 **Premium Experience**
| Feature | Description |
|---------|-------------|
| 🌙 **Dark Mode UI** | Modern, professional dark theme |
| 🎭 **Animated Elements** | Smooth transitions & visual feedback |
| 📱 **Responsive Design** | Works on all devices |
| 🎯 **User-Friendly** | Intuitive interface for HR teams |

## 🛠️ Tech Stack

**Frontend:**
- **Streamlit** - Web application framework
- **Plotly** - Interactive charts and graphs
- **Custom CSS** - Premium dark theme styling

**Backend:**
- **Python 3.8+** - Core programming language
- **SQLite** - Lightweight database
- **SMTP** - Email automation

**Libraries:**
- `pandas` - Data manipulation
- `PyPDF2` - PDF processing
- `python-docx` - DOCX file handling
- `hashlib` - Password security
